프로젝트 제목: 장내 미생물 리포터: 미생물을 기반으로 몸 균형을 유지하기 위한 웹 프로젝트


개요:
의생명시스템학부 전공 지식을 바탕으로 사용자의 식단 데이터를 분석하여  
장내 미생물 반응과 건강 개선 조언을 제공하는 웹 애플리케이션이다.
단순한 칼로리 계산이 아닌, 음식이 장내 미생물 생태계에 미치는 영향을 분석하여  
현대인의 만성 장 건강 문제 예방과 올바른 식습관 형성을 돕는 것이 목표이다.


주요 기능:
- 사용자가 입력한 식단을 분석하여 유익균/유해균 비율 계산  
- 발효식품, 채소류 → 유익균 증가 / 가공식품, 고지방식 → 유해균 증가로 반응 시뮬레이션  
- 분석 결과에 따라 AI 스타일 식단 조언 및 유산균 추천 리스트 제공  
- `localStorage`를 활용하여 이전 입력 데이터를 기억 (실험 데이터 유지)  


사용 기술 스택:
HTML, CSS, JavaScript (Vanilla)
 localStorage (브라우저 내 데이터 저장)
 Cursor IDE / Visual Studio Code  
 ChatGPT‑5 (코드 자동화 및 로직 구성 도움)
 Google Chrome  


실행 방법:
1. GitHub 저장소에서 프로젝트를 다운로드하거나 ZIP으로 받기  
2. 폴더 안의 `index.html` 파일을 더블 클릭
3. 웹 브라우저(Chrome 추천)에서 식단 입력 후 식단 분석하기 버튼 클릭  
4. 결과로  
   - 유익균 / 유해균 비율  
   - AI 조언  
   - 추천 식단, 피해야 할 음식, 유산균 정보  
   가 화면에 표시됨
